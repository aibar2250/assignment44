# Graph Traversal and Representation System

## Project Overview

This project demonstrates graph representation and traversal algorithms using Java.  
The graph is implemented using an adjacency list structure.

The project includes:
- Vertex representation
- Edge representation
- Graph implementation
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Performance testing using different graph sizes

Graphs are important data structures used in computer science for representing connections between objects.

A graph consists of:
- Vertices (nodes)
- Edges (connections between vertices)

The project tests traversal algorithms on graphs with:
- 10 vertices
- 30 vertices
- 100 vertices
